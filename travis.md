## TRAVIS SETUP

```yml
# language
language: node_js
# node version
node_js:
  - 12
# before install command
before_install:
  - npm install -g npm
# scripts to run
script:
  - npm install
  - npm test
# deploying to heroku
# generate automatic with travis setup heroku
# encryption with travis encrypt $(heroku auth:token) --add deploy.api_key --org
deploy:
  provider: heroku
  api_key:
    secure: api_key_autogenerated_by_travis
  app: app_name_on_heroku
  on:
    repo: repo OWNER/NAME
```

## ALL COMMANDS USED
```bash
$ travis setup heroku
```
